MiniLLM
トランスフォーマーベースの超小型言語モデル（日本語コメント付き）
📌 プロジェクト概要
本リポジトリは、PyTorch で実装された ** 超小型 LLM（MiniLLM）** です。高校生が AI・ディープラーニングを学ぶために作成された、シンプルかつ可読性の高い実装を目指しました。
マルチヘッド自己注意機構、残差接続、レイヤー正規化といった LLM のコア機構を、最小限のコードで再現しています。
✨ 主な特徴

🧠 軽量設計：ローカルの PC でも簡単に実行可能
📖 高可読性：詳細な日本語コメントを付与
🛠️ 機能完備：テキスト生成のデモ付き
🎓 初心者向け：LLM の仕組みを理解するのに最適

📂 ファイル構成plaintextMiniLLM/
├── mini_llm.py    # メインモデルコード（日本語コメント付き）
├── requirements.txt  # 依存関係
└── README.md      # 本ファイル

🔧 実行環境

Python 3.8 以上
PyTorch 2.0 以上

🚀 インストール方法

リポジトリをクローンbash运行git clone https://github.com/your-username/MiniLLM.git
cd MiniLLM

依存パッケージをインストールbash运行pip install -r requirements.txt


💻 実行方法bash运行python mini_llm.py

実行すると、モデルの初期化とサンプルテキスト生成が行われます。
📊 モデル構成

レイヤー数：2
隠れ次元：128
アテンションヘッド数：2
語彙サイズ：512
最大シーケンス長：64
